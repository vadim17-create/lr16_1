 # Django Shop Project

## Описание
Проект представляет собой мини-сайт интернет-магазина, выполненный в рамках лабораторной работы.  
Сайт показывает страницы с информацией о магазине, авторе и содержит мини-каталог товаров.

## Страницы магазина

1. **Главная страница** (`/`)  
   - Приветствие  
   - Ссылки на страницы "О магазине" и "Об авторе"

2. **О магазине** (`/about/`)  
   - Информация о магазине  
   - Мини-каталог товаров (например, рыболовные снасти)

3. **Об авторе** (`/author/`)  
   - Информация об авторе лабораторной работы
